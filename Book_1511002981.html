Тег DOCTYPE html- нужен указания типа текущего документа.

Тег pre - нужен для определения блока предварительно форматированного текста.

Тег html - нужен для заключения в себе всех содержимых веб-страницы.

Атрибуты - это возможность расширить возможности отдельных тегов и более гибко управлять содержимым контейнеров.

Атрибут тега html lang="en" - это возможность определить на каком языке был оформлен текст.

Одинарный тег- это однин апостроф перед и после словом ('пример').

Двойной(закрывающий) тег - это это два апострофа перед и после словом ("пример").

Тег head - нужен для хранения других элементов, цель которых — помочь браузеру в работе с данными.

Вложенный тег meta - нужен для определения метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем.

Вложенный тег title - нужен для определения заголовка документа.

Тег Header - нужен для задавания «шапки» сайта или раздела, в котором обычно располагается заголовок.

Вложенный тег H1 - нужен для того-что показывают относительную важность секции.

Тег body - нужен для хранения содержания веб-страницы (контента), отображаемого в окне браузера.

Тег Вложенный тег P - нужен для определения текстовых абзацов.

Тег footer - нужен для создания «подвала» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
